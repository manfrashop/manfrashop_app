        🗑️ Réinitialisation sélective des données

        Supprime définitivement les données locales sélectionnées. Les données Firebase/Firestore ne sont pas affectées.

        📦 Remettre tous les stocks à zéro

        Remet la quantité en stock (p.s) à 0 pour tous les produits du catalogue local. N'affecte pas Firestore.

        ⚠️ Vider le catalogue & clients hardcodés

        Vide les données STATIC_DB chargées au démarrage (produits, clients, fournisseurs). L'application redémarrera avec un catalogue vierge. Rechargez la page pour confirmer l'état.

        ☢️ Réinitialisation locale

        Efface toutes les données locales (localStorage manfara_*) ET vide le STATIC_DB en mémoire. Action irréversible.

        🔥 REMISE À ZÉRO TOTALE — Firebase + Local

        Supprime définitivement toutes les données :

        • Collections Firestore : produits, clients, fournisseurs, ventes, achats, depenses, appros, stock_mvt, stock_inv, points_de_vente, credits_remboursements, clotures, retours
        • localStorage complet manfara_*
        • Toutes les variables en mémoire
        ⚠️ Cette action est IRRÉVERSIBLE. Les comptes utilisateurs Firebase Auth ne sont pas supprimés. Faites un backup avant de continuer.
